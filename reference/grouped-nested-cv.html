Group splitters

from sklearn.model_selection import (
    GroupKFold, StratifiedGroupKFold, cross_val_score)

gkf = GroupKFold(n_splits=5)
scores = cross_val_score(clf, X, y, groups=entity_id, cv=gkf)

# class balance + group isolation
sgkf = StratifiedGroupKFold(n_splits=5, shuffle=True, random_state=42)
scores = cross_val_score(clf, X, y, groups=entity_id, cv=sgkf)

Rules: pass groups (or it errors); folds are unequal in size; need ≥ n_splits distinct groups.

Nested CV (unbiased tuned estimate)

from sklearn.model_selection import GridSearchCV, cross_val_score, KFold

inner = KFold(5, shuffle=True, random_state=1)   # picks hyperparameters
outer = KFold(5, shuffle=True, random_state=2)   # scores the procedure

search = GridSearchCV(clf, param_grid, cv=inner)
nested = cross_val_score(search, X, y, cv=outer)
report = nested.mean()   # the honest number

Two loops, two jobs

LoopQuestionKeep
InnerWhich hyperparameters?best params per fold
OuterHow well does it generalize?mean ± std (headline)

Report the outer mean, not GridSearchCV.best_score_ — the latter is selection-biased (Cawley & Talbot, JMLR 2010).

Grouped + nested

inner = GroupKFold(5); outer = GroupKFold(5)
search = GridSearchCV(clf, param_grid, cv=inner)
nested = cross_val_score(search, X, y, cv=outer,
                         params={"groups": entity_id})  # sklearn >=1.4

Mistakes → fixes

MistakeFix
Row-level K-fold on grouped dataGroupKFold
GroupKFold on imbalanced labelsStratifiedGroupKFold
Reporting best_score_Nested CV; report outer mean
Dropping groups in outer loopRoute groups to both loops
