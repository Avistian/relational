Notebooks

Read a lab in the browser, or run it live — no local install needed.

Every lab has four options:

  • View — the notebook rendered as a static HTML page (read-only, instant).
  • Source .ipynb — the raw notebook to download or open in your own Jupyter.
  • Run on Binder — launches a free cloud Jupyter with the real environment (the relkit harness + live OpenML fetch). First launch takes a minute to build; after that the CHECK/EXIT cells run exactly as locally.
  • Open in Colab — opens the notebook in Google Colab. Just run the first cell (Running on Google Colab?): it clones the repo, installs requirements-labs.txt, and switches into labs/ so relkit and the data cache work. No manual setup needed.
